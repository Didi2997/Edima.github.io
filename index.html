DATA CREATION, MODELLING, AND EXPLORATORY ANALYSIS IN SQL

A bunch of T-SQL Queries written for the purpose of Data Modelling and Exploratory Analysis of the PLATINUM READERS database. The modelling of this database sees the creation and use of SEQUENCES and DEFAULT FORMATTING to customize and auto-increment the identity columns of two out of the four tables in this Database. Also, there's the inclusion of CHECK, UNIQUE, NOT NULL constraints, as well as PRIMARY and FOREIGN KEYS to secure and maintain data integrity. This database comprises four tables which contain information about book sales, author information and customer data. The queries deployed for the purpose of Data exploration include aggregate functions (precisely MIN, MAX, AVG, SUM), UNION and EXCEPT operators, Various JOINS, CASE statements, FORMAT functions, GROUP BY, ORDER BY and WHERE clauses, WINDOWS functions (including PARTITION BY, ROW NUMBER, UNBOUNDED PRECEDING, AND DENSE RANK), CTEs, Subqueries, Derived Tables, and Temp Tables.
